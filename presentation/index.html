Challenge #181

DOM Clobbering

Saviez-vous qu’en JavaScript, lorsqu’une variable n’est pas définie, le moteur peut automatiquement aller chercher un élément HTML du DOM portant le même identifiant (id) ? Ce comportement peut entraîner des effets de bord inattendus… voire être exploité pour détourner la logique d’un script !
Vous devez réussir à activer votre compte !

Pour cela vous devez avoir les connaissances suivantes :
